ACME Engineering Space Systems – Launch and Missile Defense Systems has an exciting career opportunity for a Principal Software Integration Engineer to join our team of qualified, diverse individuals. This position will be located in Huntsville, AL.

Position Overview:

  

The selected Principal Software Integration Engineer will provide support for the Command and Control, Battle Management, and Communications (C2BMC) program. The selected candidate will support the C2BMC Integration team to integrate complex applications on Windows and Linux platforms for a critical customer program.

Essential Functions:

  • Installation and configuration of Linux and Windows HS platforms from the OS up to the application level
  • Documentation and verification of all installation and configuration steps to be delivered to the customer
  • Develop test plans procedures for software integration test conduct
  • Be responsible for integration/execution, detailed data analysis and preparation of software test reports
  • Coordinate execution of SW integration tests to verify system operation
  • Work with software developers to diagnose the root cause of problems found during integration

.

Occasional travel will be required including support to installation and checkout at customer sites (including OCONUS).

The selected candidate should thrive in a fast-paced work environment with high expectations, significantly diverse assignments, collaborative/team settings across all levels.

BQs for a Principal Software Integration Engineer:

  • Bachelor’s Degree in a STEM (Science, Technology, Engineering or Mathematics) discipline preferred from an accredited university and 5 years of related experience, or a Master's Degree and 3 years of related experience, or 0 years of experience with a PhD.
  • Previous experience with integration of separate software components into a single system
  • Experience with Windows and Linux system administration
  • Experience with administering and installing VMWare systems
  • Candidate must have, or be able to develop, in depth expertise in the C2BMC concept of operations and user interface
  • Experience writing installation manuals
  • Ability to apply IA requirements to an existing system
  • CompTIA Security+ (or equivalent) DoD 8570.01M IAT Level II – Certification
  • A current active in-scope U.S. Government DoD Secret security clearance is required to start.

Preferred Qualifications:

  • Experience with Ansible Automation Platform, including Ansible Tower
  • Experience with VM Horizon View
  • Familiarity with Java and object oriented development
  • Ability to read and troubleshoot application source code
  • A current active in-scope U.S. Government DoD Top Secret security clearance.
